# C Guide

Livro completo de Linguagem **C** em pt-BR — do hello world ao C moderno (C23).

🔗 **Live:** https://wallysondevs.github.io/c-guide/

## Estrutura

28 capítulos divididos em blocos:
- **Fundamentos** — história, compilação, tipos, operadores, controle de fluxo
- **Estrutura do código** — funções, escopo, arrays, strings
- **Ponteiros** — bloco completo (4 capítulos) com diagramas de memória
- **Tipos compostos & organização** — structs, pré-processador, headers, make
- **Standard Library** — stdio, stdlib, string, math, time
- **Profissional** — undefined behavior, gdb, C moderno, threads, projeto final

Cada capítulo segue o ritmo: **analogia → diagrama → exemplo do mundo real → antes/depois → armadilhas → resumão**.
