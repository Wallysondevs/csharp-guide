# C# - Guia Completo do Zero ao Avançado

Livro online, didático e profundo, sobre C# / .NET — escrito para quem está começando do zero absoluto e quer chegar até tópicos avançados (LINQ, async, EF Core, ASP.NET Core, performance).

## Stack
- Vite + React + TypeScript
- Tailwind CSS v4
- Wouter (hash routing)
- Deploy GitHub Pages: branch `gh-pages`

## Local
```
pnpm install
pnpm dev
```
